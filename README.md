# IotFramework

Nestjs Monorepo with NX

- Docker
- MQTT broker
  - Eclipse mosquitto
- PostgreSQL
- TypeORM
- Redis
- Nestjs
- NX

## Setup

```
git clone https://github.com/KGESH/iot-framework.git
```

```
cd iot-framework
```

```
npm install
```

## Start local
run docker engine. before npm start  

```
npm start
```
